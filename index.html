Omnicatz Inventory System

A simple system using a standard barcode scanner and dymo labelmanager to keep a database of for tracking object and containers.

The idea is to only track where an item was last... One never actually track when one remove an item unless it purged totally from the system.

Label Printer

Barcode scanner

Additionally I am working on a Phone app mostly for simpler operations like moving items that already exist to another container also I want to add functionality so it can substitute the barcode scanner.

To do this I am integrating a http server with the PC app so it serves as a server probably running in the tray later on and with an option to launch on start.

I am also planing on a low budget option of pre-printing barcodes using a regular printer where by one would print a whole bunch of them at a time then cut them out as needed and glue them on.
